# Matrix Spiral Order Traversal

The function spiralOrderMatrix(matrix) takes a 2D square matrix (N Ã— N) as input and prints its elements in a spiral order, starting from the top-left corner and moving right, down, left, and up in a circular pattern until all elements are traversed. Prints error for invalid input.

## Command

To execute

```sh
node matrix-spiral-order.js
```
